AI 论文解析报告生成系统 - 软件使用说明书
版本： 1.0
最后更新日期： 2025年1月15日

1. 系统简介
欢迎使用“AI 论文解析报告生成系统”！本系统旨在利用先进的文档智能解析（MinerU）与大语言模型（LLM）技术，为科研工作者提供一站式的学术论文分析服务。您只需上传PDF格式的学术论文，系统即可自动为您生成结构化的导读报告，并支持基于报告内容的智能问答和多篇论文的对比分析，极大地提升您的文献阅读与研究效率。

本系统采用B/S（浏览器/服务器）架构，您只需通过标准的Web浏览器即可访问全部功能，无需安装任何额外的客户端软件。

2. 系统要求
操作系统: Windows, macOS, Linux 均可。
浏览器: 推荐使用最新版本的 Google Chrome, Microsoft Edge, Firefox 或 Safari。为获得最佳体验，请确保您的浏览器已启用JavaScript。
网络: 需要稳定的互联网连接，用于上传文件、调用AI服务及与后端服务器通信。
文件格式: 支持上传 .pdf 格式的文件。
3. 快速开始
3.1 访问系统
在您的浏览器地址栏中输入系统的URL（http://8.136.43.23:80），即可进入系统首页。

3.2 用户注册与登录
首次使用本系统，您需要先进行注册。

注册新账户:
在登录页面，点击 “立即注册” 链接。
在注册表单中，填写 用户名、密码 并再次确认密码。
点击 “注册” 按钮。注册成功后，系统将自动跳转至主功能界面。

用户登录:
在登录页面，输入您的 用户名 和 密码。
点击 “登录” 按钮。登录成功后，系统将根据您的身份（普通用户或管理员）跳转至相应的主界面。
注意: 管理员账户为 Admin，初始密码为 621control。普通用户注册时，用户名不能为 Admin或admin。

3.3 主界面概览 
登录成功后，您将进入主功能界面。界面主要由以下几个区域组成：

左侧侧边栏: 包含 上传论文、预览编辑、AI问答、任务列表 四个导航按钮，以及一个 登出 按钮。
中央工作区: 根据您点击的侧边栏按钮，动态显示对应的功能模块。
顶部标题: 显示当前页面的名称。

4. 功能详解
4.1 上传与解析论文
此功能是整个系统的核心入口，用于创建新的“导读任务”。

选择文件:
点击侧边栏的 “上传论文” 按钮，或滚动到页面顶部的 “上传论文 PDF” 区域。
点击 “选择文件” 按钮，在弹出的文件选择对话框中，选择一个或多个 .pdf 文件。本系统支持多文件同时上传。
创建任务:
选中文件后，它们会出现在 “上传任务” 列表中。您可以随时点击 “移除” 按钮删除不需要的文件。
当您准备好后，点击 “作为任务分析” 按钮。
系统会弹出一个对话框，要求您为本次上传的论文集合命名一个 任务标题（例如：“CVPR 2024 图像分割论文合集”）。输入标题后点击确定。
等待分析:
提交后，系统将开始处理您的请求。您会看到一个进度条，显示从“准备任务”到“服务器正在处理文件”再到“任务完成”的全过程。
此过程包括：向MinerU服务发送PDF进行结构化解析，然后调用大语言模型（LLM）对每篇论文生成一份AI导读报告。
分析完成后，第一篇论文的报告将自动加载到 “论文内容” 区域。
4.2 查看与管理任务
所有您创建的“导读任务”都会被保存下来，方便您随时回顾。

查看任务列表:
点击侧边栏的 “任务列表” 按钮，或滚动到页面底部的 “任务列表” 区域。
您将看到一个表格，列出所有任务的 序号、任务ID、标题、包含的论文数量、状态 以及 创建时间。
任务操作:
对于列表中的每个任务，您可以执行以下操作：
查看详情: 点击 “查看详情” 按钮，会弹出一个模态框，展示该任务下所有论文的详细信息（如文件名、上传时间）以及已生成的报告列表。
查看报告: 点击 “查看报告” 按钮，系统会提示您选择该任务下的某一份报告进行加载。选中后，报告内容将显示在 “论文内容” 区域。
多论文对比: 点击 “多论文对比” 按钮，系统将跳转到专门的 多论文对比页面（详见 4.5 节）。
删除任务: 点击 “删除任务” 按钮，系统会弹出确认对话框。确认后，该任务及其关联的所有论文和报告将被永久删除。
4.3 报告预览与编辑
当一篇论文的报告被加载后，您可以在 “论文内容” 区域对其进行查看和编辑。

切换模式:
预览模式: 默认模式，以格式化的方式展示Markdown报告，数学公式会被正确渲染。
编辑模式: 点击 “切换编辑/预览” 按钮，进入编辑模式。此时，您将看到原始的Markdown文本，可以对其进行任意修改。
保存报告:
在预览或编辑模式下，如果内容有改动，“保存报告” 按钮将变为可用状态。
点击 “保存报告” 按钮，系统会弹出一个对话框，列出您所有的任务。
请选择一个任务，将当前报告关联到该任务下。保存成功后，您可以在该任务的详情中找到这份报告。
下载报告:
点击 “下载报告” 按钮，当前显示的报告内容将以 .md (Markdown) 格式文件的形式下载到您的本地计算机。
4.4 AI智能问答
您可以就当前加载的论文报告内容向AI提问。

发起提问:
向下滚动到 “AI 问答” 区域。
在输入框中输入您的问题（例如：“这篇论文的主要创新点是什么？”）。
点击 “提问” 按钮。
查看回答:
您的问题和AI的回答会以聊天记录的形式显示在上方的消息区域中。AI的回答基于您当前加载的论文内容，力求准确和相关。
4.5 多论文对比分析
此功能允许您对一个任务下的所有论文进行综合性的对比分析。

进入对比页面:
在主界面的任务列表中，点击任一任务的 “多论文对比” 按钮，系统将跳转至多论文对比页面，并自动携带任务ID。
输入对比提示:
在 “输入对比提示” 文本框中，清晰地描述您希望AI从哪些角度进行对比（例如：“请从方法论、实验设计和最终性能三个方面，对比这三篇论文的异同点和优劣。”）。
生成对比报告:
点击 “开始对比” 按钮。系统将调用LLM对任务内的所有论文进行综合分析。
分析完成后，一份结构化的 对比报告 将显示在下方区域。
返回主界面:
阅读完报告后，您可以点击 “返回” 按钮，回到 main.html 的任务列表页面。
4.6 管理员功能 
管理员拥有特殊的权限，可以审查所有用户的数据。

访问管理员面板:
使用管理员账户 (Admin / 621control) 登录后，系统将自动跳转至 管理员面板。
用户审查:
在 “所有用户列表” 表格中，您可以看到系统内所有用户的 ID 和 用户名。
点击任一用户的 “查看用户界面” 按钮，系统将在新标签页中以该用户的身份打开主界面。您可以像该用户本人一样，查看其所有的任务和报告。
返回管理员界面:
在审查用户的主界面左侧，会显示一个 “返回管理员界面” 按钮。点击此按钮即可关闭当前标签页，回到 管理员面板。
登出:
点击左下角的 “登出” 按钮，即可退出管理员账户。

5. 常见问题 (FAQ)
Q1: 上传PDF后，长时间处于“服务器正在处理文件”状态怎么办？

A: 论文解析和AI生成报告是一个计算密集型任务，耗时取决于PDF的复杂度和服务器负载。请耐心等待。如果超过10分钟仍未完成，可尝试刷新页面或联系系统管理员。

Q2: 为什么有时无法加载报告或对比报告为空？

A: 这可能是由于AI服务暂时不可用或PDF内容过于复杂导致解析失败。请检查您的网络连接，或尝试上传其他PDF文件。

Q3: 我不小心删除了任务，还能恢复吗？

A: 目前系统不提供数据恢复功能。删除操作是永久性的，请在执行删除前务必确认。
